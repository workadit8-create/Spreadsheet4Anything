# Dokumen operasional — Akuntansi App

Semua file Word siap download. Regenerate setelah edit script:

```bash
./scripts/generate-all-docs.sh
```

| Dokumen | File | Untuk apa |
|---------|------|-----------|
| Mitigasi & penanganan bugs | `Mitigasi-Penanganan-Bugs-Akuntansi-App.docx` | Alur fix bug, SLA, template balas client |
| Onboarding client baru | `Onboarding-Client-Baru-Akuntansi-App.docx` | Setup client 2, 3, … |
| Smoke test 5 menit | `Checklist-Smoke-Test-5-Menit.docx` | Setelah setiap deploy |
| Form lapor bug | `Template-Form-Lapor-Bug.docx` | WhatsApp template + struktur Google Form |
| Early Partner (5 client pertama) | `Early-Partner-Management.docx` | Kesepakatan pilot, tracker, transisi harga |
| Tracker spreadsheet | `Early-Partner-Tracker.csv` | Import ke Google Sheet |

Buka semua di Finder:

```bash
open docs/
```
